# Calculator Web App

A simple and stylish calculator built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic frontend development skills and logic building without relying heavily on built-in functions.

---

## 🚀 Features

- ➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- 🧹 Clear display functionality
- ⌫ Delete last digit (implemented using manual logic)
- 🎨 Modern dark UI with responsive design
- ⚡ Fast and lightweight (no frameworks used)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)

---

## 💡 Key Highlights

- Implemented calculator logic using **pure JavaScript**
- Avoided built-in string functions for delete operation (manual loop logic)
- Clean and user-friendly interface design
